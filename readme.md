# Two Player Carry System

A lightweight Roblox prototype demonstrating a synchronized two-player "carry" mechanic. Players can attach to opposite ends of a shared part and move together, simulating cooperative carrying behaviour.

A quick prototype demonstrating a two-player "carry" mechanic using spring constraints, attachments and align orientation on Roblox Studio. Players can attach to opposite ends of a shared part and move together, simulating cooperative carrying behavior.

Demonstation video: https://youtu.be/1P0Wx-O2Crw

# Installation
There are 2 ways to install this into your device:
1. Download the .rbxl file in the repository and open it with Roblox Studio
2. Install Pesto, a VS Code to Roblox Studio Sync Tool and import it into Roblox Studio using its plugin
